d1pythonitk - Python DataONE ITK
================================

Client tool in python for interacting with DataONE_ services exposed by Member 
and Coordinating Nodes.

To build the documentation::

  cd $HOME/Projects/D1_cicore/
  export CDIR=`dirname $PWD`
  export PYTHONPATH="$CDIR/api-common-python:$CDIR/itk/d1-python/src:$PYTHONPATH"
  cd itk/d1-python/doc
  make html


.. _DataONE: http://dataone.org
