Documentation for d1pythonitk - A DataONE Client in Python
==========================================================

d1pythonitk is a DataONE client written in Python. d1pythonitk provides programmatic access
to the DataONE infrastructure that may be used as a command line tool for
interacting with DataONE services, may form the basis of larger applications,
or may be used to extend existing applications to utilize the services of
DataONE.

d1pythonitk is a part of the DataONE Investigator Toolkit (ITK), and is developed as
part of the `DataONE project`_.
