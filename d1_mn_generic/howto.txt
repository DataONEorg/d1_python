Misc
----

Create empty db:
rm *sq3
./manage.py syncdb

Run Django dev server:
./manage.py runserver

Run unit tests:
./manage.py test mn_service


Fixtures
--------

Create fixture for empty db:
echo >empty.fixture.json []

Create fixture for db containing current data:
rm *sq3
./manage.py syncdb
./manage.py runserver
curl -X GET -H "Accept: application/json" http://127.0.0.1:8000/mn/update/
./manage.py dumpdata mn_service >base.fixture.json
