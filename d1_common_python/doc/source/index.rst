Documentation for d1_common_python - DataONE Common
===================================================

``d1_common_python`` is a DataONE client library written in Python. It provides
functionality commonly needed by projects that interact with the DataONE
infrastructure via Python. It is a dependency of ``d1_libclient_python``,
``Generic Member Node`` and all other DataONE components written in Python.

``d1_common_python`` is a part of the DataONE Investigator Toolkit (ITK), and
is developed as part of the `DataONE project`_.
